Research Projects

AI-Enhanced Remote Sensing for Archaeological Discovery

My ongoing research applies advanced machine learning algorithms to satellite imagery for the identification of archaeological sites, with a focus on raised field structures in the Llanos de Moxos area in Beni, Colombia. This project bridges cutting-edge AI techniques with traditional archaeological methods to enable non-invasive site detection.

Technical Approach

This research employs a comprehensive technical framework that combines multiple satellite imagery sources with deep learning models:

  • Multi-Source Satellite Imagery: Leveraging both Sentinel and Landsat imagery accessed through ArcGIS, providing multi-spectral data at different resolutions to identify subtle terrain features
  • YOLO Object Detection: Implementing YOLOv8 (You Only Look Once) models for their efficiency in detecting archaeological features across large geographic areas
  • Data Processing Pipeline: Custom Python and R scripts for data normalization, format conversion between KITTI and YOLO frameworks, and batch processing of satellite imagery
  • High-Performance Computing: Utilizing UFS supercomputer resources for hyperparameter grid search optimization to fine-tune model performance

Pattern Recognition Methodology

The research employs specialized pattern recognition techniques tailored to archaeological feature detection:

  • Template Matching: Algorithmic comparison of image segments against known archaeological feature templates
  • Orientation Analysis: Differentiation between ancient and modern structures based on alignment patterns:
    • Ancient structures: Often aligned with solar or celestial orientations
    • Modern structures: Typically aligned with contemporary infrastructure
  • Geometric Feature Focus: Particular attention to rectangular lakes and rectilinear patterns, which often indicate human modification in the Llanos de Moxos landscape
  • Feature Typology Development: Creating a comprehensive classification system of structure types to enhance model training and validation

Technical Implementation

The project workflow includes several sophisticated technical components:

  • Image Annotation Process:
    • Manual annotation of verified archaeological features
    • Export to deep learning formats using specialized plugins
    • Processing and augmentation using Python/R libraries
  • Coordinate Normalization: Transformation of geographic coordinates and standardization for model compatibility
  • Model Architecture: Implementation of both convolutional neural networks (CNNs) for spatial pattern recognition and recurrent neural networks (RNNs) for temporal analysis of land use changes
  • Clustering Algorithms: Application of unsupervised learning techniques to identify potential site clusters and settlement patterns

Integration with Archaeological Knowledge

This technical approach is grounded in archaeological expertise about the region:

  • Incorporation of known archaeological site characteristics specific to pre-Columbian raised field agriculture in the Amazon basin
  • Validation of model results against previously documented sites to refine detection accuracy
  • Collaboration with archaeologists specializing in Amazonian cultures to ensure AI detections align with cultural and historical contexts

Broader Applications

The methodologies developed in this project have applications beyond the immediate research:

  • Creation of comprehensive visual analyses highlighting critical conservation areas
  • Development of geo-temporal environmental references to track landscape changes over time
  • Potential application to other archaeologically rich regions with similar environmental conditions
  • Contributing to digital heritage preservation through detailed documentation of archaeological landscapes

This research represents a significant step forward in non-invasive archaeological prospection, allowing for the identification of potential sites while minimizing physical disturbance to these culturally and historically significant areas.
